# GemSync Chrome Extension

This extension adds floating Gemini study controls:

- `PDF`: open or close the side PDF panel.
- `Restore`: jump back to a saved Gemini reading position.
- `Mark`: save the current Gemini reading position.
- `Top`: manually load older Gemini messages toward the beginning of the chat.
- `Stop`: cancel an active loading/search action.

The PDF panel reads local generated config files under:

```text
extension/pdf-panel/subjects.json
extension/pdf-panel/subjects/
```

Those files are generated by GemSync Manager and are ignored by git because they contain local PDFs and Gemini conversation links.

## Install

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click **Load unpacked**.
4. Select the `extension` folder.
5. Refresh Gemini.

## Generate PDF Panel Data

Use the manager web UI and click **Write To Extension** after your course screenshots and Gemini conversations are ready.
